val make_board : unit -> Board.board

make_board initializes a board for the game with a list of player names.

val get_player_name_input : Board.board -> string

get_player_name_input board is the name inputted.

val ask_for_money : 'a -> Player.player -> Player.player -> int -> int -> unit

ask_for_money asks from_player which money cards they want to give to current_player such that the total value is at least total_value or they are out of cards, and transfers it from their piles.

val pass_go : Board.board -> bool

pass_go board distributes up to 2 cards to the current player.

val its_my_bday : Board.board -> bool

its_my_bday board gets $2M from every other player and transfers it to the current player's pile.

val swap_out : Player.player -> string -> Card.card
val move_property : Board.board -> bool -> string -> bool

move_property moves a property from player with name name to the current player's pile.

val sly_deal : Board.board -> bool

sly_deal board allows the current player to steal any property of their choice from any other player of their choice.

val forced_deal : Board.board -> bool

forced_deal board allows the current player to switch any of their properties with any other player's property of their choice.

val debt_collector : Board.board -> bool

debt_collector board allows the current player to request $5M from any other player of their choice.

val deal_breaker : Board.board -> bool
val action_card_helper : Board.board -> int -> bool

action_card_helper performs the corresponding functionality of the action card with id id.

val play_helper : Board.board -> int -> int -> unit

play_helper performs the card actions when the player uses the 'play' command.

val main_helper : Board.board -> int -> unit

main_player board num represents a pass of each command input. A player can only play 3 cards per turn.

val main : unit -> unit

main starts the game.
