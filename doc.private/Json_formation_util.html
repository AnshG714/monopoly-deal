val string_input_helper : unit -> string

string_input_helper is the string that the user inputs in the terminal. An important thing to remember is that if you press enter by mistake, nothing will be taken in, and the function will default to an empty string. This is intentional. If this was a mistake and you want to change this, you have to do so manually in card_data.json

val int_input_helper : unit -> int

int_input_helper is the integer that the user inputs in the terminal. An important thing to remember is that if you press enter by mistake, nothing will be taken in, and the function will default to 0. This is intentional. If this was a mistake and you want to change this, you have to do so manually in card_data.json

val continue_check_helper : (unit -> unit) -> unit

continue_check_helper f will call the function f if the user input is 'y' or 'Y', or it will simply return unit (thus exiting)
